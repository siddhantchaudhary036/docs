All widget settings are configured from **Settings > Widget** in your dashboard. Changes take effect immediately for all users.

## Position

Choose where the widget launcher appears:

| Option | Description |
|---|---|
| Bottom-right | Default. Launcher in the bottom-right corner. |
| Bottom-left | Launcher in the bottom-left corner. |

## Launcher style

### Button launcher (default)

A floating circular button with an icon. Choose from these icons:

- Chat bubble
- Bell
- Megaphone
- Gift

### Tab launcher

A fixed tab on the side of the screen with a text label. Configure:

- Tab label (e.g., "What's New")
- Tab side (left or right)

## Colors

| Setting | Description |
|---|---|
| Primary color | Main accent color for the widget |
| Badge color | Color of the unread count badge |
| Color scheme | Light, dark, or auto (follows system preference) |

## Appearance

| Setting | Description |
|---|---|
| Header text | Text shown at the top of the widget panel (default: "What's New") |
| Border radius | Roundness of the widget panel corners |
| Show branding | Toggle ReleaseDock branding in the widget footer |
| Show subscribe | Toggle the email subscribe form inside the widget |

## Preview

Use the widget preview in **Dashboard > Widget** to see how your configuration looks before it goes live. The preview shows the launcher button and the open panel with sample changelog entries.
